## Preguntas frecuentes

### ¿Cómo escaneo una nota?

Toca **Start Your Patch** en la pantalla principal, elige **Scan Document** para capturar con la cámara, o **Choose from Library** para importar fotos existentes.

### ¿Cómo ajusto el tamaño de impresión?

Usa el control deslizante de ancho en el editor para definir el tamaño exacto. Si el resultado impreso tiene alguna desviación, usa la herramienta de **Calibración** en Ajustes: imprime la línea de referencia, mídela con una regla e introduce la medida real. Solo necesitas calibrar una vez.

### ¿Qué es Smart Text?

Smart Text extrae texto de imágenes: notas manuscritas, páginas impresas, fotos o diseños mixtos. Dos modos disponibles:

- **OCR en dispositivo:** Gratuito, funciona sin conexión. Usa el framework Vision de Apple.
- **Reconocimiento IA (Pro):** Mayor precisión para caligrafía difícil, fórmulas, tablas y contenido mixto. Envía la imagen a una API en la nube segura.

### ¿NotePatch sube mis datos?

Las funciones principales (escaneo, recorte, máscara, impresión) son **completamente offline**. Los datos solo salen de tu dispositivo cuando:

- Imprimes en una impresora que selecciones (por red local)
- Compartes un PDF con la función de compartir de iOS
- Usas el reconocimiento IA (Pro), que envía la imagen a una API en la nube — las imágenes no se almacenan ni se usan para entrenamiento

### ¿Qué es NotePatch Pro?

Una suscripción opcional que desbloquea el reconocimiento de texto con IA y futuras funciones premium. Las funciones principales son gratuitas para siempre. Gestiona o cancela en Ajustes → Apple ID → Suscripciones.

### ¿Cómo restauro mi suscripción?

Ve a Settings → NotePatch Pro → Restore Purchases.

### ¿Qué dispositivos son compatibles?

iPhone y iPad con iOS 17.6 o posterior.
