## 常见问题

### 怎么扫描笔记？

在首页点击"开始制作"，选择"扫描文档"用相机拍摄，或选择"从相册选择"导入已有照片。

### 怎么让打印尺寸刚好？

在编辑器中使用底部的宽度滑块设定精确尺寸。如果打印结果有偏差，到设置中使用"校准"工具——打印参考线，用尺子量一下，输入实际长度即可。只需校准一次。

### 什么是智能文字？

智能文字可以从图片中提取文字——手写笔记、印刷文档、照片或图文混排内容。两种模式：

- **本地识别：** 免费，离线可用。使用 Apple Vision 框架。
- **AI 识别（Pro）：** 更高精度，适合潦草字迹、公式、表格等复杂内容。将图片发送到安全的云端 API 处理。

### NotePatch 会上传我的数据吗？

核心功能（扫描、裁剪、遮罩、打印）**完全离线**。数据仅在以下情况下离开设备：

- 打印到你选择的打印机（通过局域网）
- 通过 iOS 分享功能分享 PDF
- 使用 AI 识别（Pro），将所选图片发送到云端 API 提取文字——图片不会被存储或用于训练

### NotePatch Pro 是什么？

可选订阅，解锁 AI 文字识别等高级功能。核心功能永久免费。可在"设置 → Apple ID → 订阅"中管理或取消。

### 怎么恢复已购订阅？

进入设置 → NotePatch Pro → 恢复购买。

### 支持哪些设备？

iPhone 和 iPad，需要 iOS 17.6 或更高版本。
