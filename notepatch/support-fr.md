## Questions fréquentes

### Comment scanner une note ?

Appuyez sur **Start Your Patch** sur l'écran d'accueil, choisissez **Scan Document** pour capturer avec l'appareil photo, ou **Choose from Library** pour importer des photos existantes.

### Comment ajuster la taille d'impression ?

Utilisez le curseur de largeur dans l'éditeur pour définir la taille exacte. Si le résultat imprimé présente un écart, utilisez l'outil de **Calibration** dans Réglages : imprimez la ligne de référence, mesurez-la avec une règle et saisissez la mesure réelle. Une seule calibration suffit.

### Qu'est-ce que Smart Text ?

Smart Text extrait du texte à partir d'images : notes manuscrites, pages imprimées, photos ou mises en page mixtes. Deux modes disponibles :

- **OCR sur l'appareil :** Gratuit, fonctionne hors ligne. Utilise le framework Vision d'Apple.
- **Reconnaissance IA (Pro) :** Plus grande précision pour les écritures difficiles, formules, tableaux et contenus mixtes. Envoie l'image vers une API cloud sécurisée.

### NotePatch envoie-t-il mes données ?

Les fonctions principales (scan, recadrage, masquage, impression) sont **entièrement hors ligne**. Les données ne quittent votre appareil que lorsque vous :

- Imprimez vers une imprimante sélectionnée (via le réseau local)
- Partagez un PDF via la fonction de partage iOS
- Utilisez la reconnaissance IA (Pro), qui envoie l'image vers une API cloud — les images ne sont ni stockées ni utilisées pour l'entraînement

### Qu'est-ce que NotePatch Pro ?

Un abonnement optionnel qui débloque la reconnaissance de texte par IA et de futures fonctionnalités premium. Les fonctions principales restent gratuites pour toujours. Gérez ou annulez dans Réglages → Apple ID → Abonnements.

### Comment restaurer mon abonnement ?

Allez dans Settings → NotePatch Pro → Restore Purchases.

### Quels appareils sont compatibles ?

iPhone et iPad sous iOS 17.6 ou ultérieur.
