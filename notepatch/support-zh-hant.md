## 常見問題

### 怎麼掃描筆記？

在首頁點擊「開始製作」，選擇「掃描文件」用相機拍攝，或選擇「從相簿選擇」匯入已有照片。

### 怎麼讓列印尺寸剛好？

在編輯器中使用底部的寬度滑桿設定精確尺寸。如果列印結果有偏差，到設定中使用「校準」工具——列印參考線，用尺量一下，輸入實際長度即可。只需校準一次。

### 什麼是智慧文字？

智慧文字可以從圖片中擷取文字——手寫筆記、印刷文件、照片或圖文混排內容。兩種模式：

- **本機辨識：** 免費，離線可用。使用 Apple Vision 框架。
- **AI 辨識（Pro）：** 更高精度，適合潦草字跡、公式、表格等複雜內容。將圖片傳送到安全的雲端 API 處理。

### NotePatch 會上傳我的資料嗎？

核心功能（掃描、裁切、遮罩、列印）**完全離線**。資料僅在以下情況下離開裝置：

- 列印到你選擇的印表機（透過區域網路）
- 透過 iOS 分享功能分享 PDF
- 使用 AI 辨識（Pro），將所選圖片傳送到雲端 API 擷取文字——圖片不會被儲存或用於訓練

### NotePatch Pro 是什麼？

可選訂閱，解鎖 AI 文字辨識等進階功能。核心功能永久免費。可在「設定 → Apple ID → 訂閱項目」中管理或取消。

### 怎麼恢復已購訂閱？

進入設定 → NotePatch Pro → 恢復購買。

### 支援哪些裝置？

iPhone 和 iPad，需要 iOS 17.6 或更高版本。
