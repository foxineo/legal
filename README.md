# foxineo/legal

Privacy policies, terms, and support pages for apps published by foxineo.

Hosted at: **<https://foxineo.github.io/legal/>**

## Apps currently covered

- **NotePatch** — `/notepatch/`
  - [Privacy Policy](https://foxineo.github.io/legal/notepatch/privacy.html)
  - [Support](https://foxineo.github.io/legal/notepatch/support.html)

## Adding a new app

1. Create a new subdirectory at the repo root: `mkdir <appname>/`
2. Add at minimum:
   - `<appname>/index.md` — landing page for the app
   - `<appname>/privacy.md` — privacy policy
   - `<appname>/support.md` — support / FAQ page
3. Each markdown file needs Jekyll front matter:
   ```yaml
   ---
   title: <Page Title>
   ---
   ```
4. Add a link to the new app in `index.md` (the site index).
5. Commit and push to `main`. GitHub Pages rebuilds automatically (~30-60 sec).

## Stack

- Static site generated by GitHub Pages (Jekyll, `jekyll-theme-cayman`)
- Configuration: `_config.yml`
- No build step needed locally — push to `main` and GitHub builds it
